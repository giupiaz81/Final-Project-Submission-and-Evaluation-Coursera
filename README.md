# Project Name: Task Manager App

## Overview
The Task Manager App is a simple project that helps users track and manage their daily tasks efficiently. It demonstrates basic CRUD operations and showcases version control using Git and GitHub.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- View all tasks
